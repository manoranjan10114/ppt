# PLAYBOOK 04 — Direct Booking Strategy

> Owner: Product + CSM + Marketing
> Target: Shift hotels from OTA dependency to direct bookings · Revenue: ₹5–10 Cr · Timeline: Q1–Q3

---

## THE CORE THESIS

Hotels pay OTAs 18–25% commission on every booking. A hotel with 100 bookings/month at ₹2,000 ADR pays ₹36,000–₹50,000/month to OTAs. If we help them get even 20% of those bookings direct, they save ₹7,200–₹10,000/month.

**Our Booking Engine is the tool that does this.** But it only works if hotels actively use it, promote it, and trust it.

This playbook is about making direct bookings the centerpiece of our value proposition — not just a feature, but the reason hotels stay, upgrade, and refer others.

### For Segment: BE Enabled but Not Active (~300 hotels)

These hotels have the tool but aren't using it. This is the lowest-hanging fruit.

**Root cause investigation (CSM calls with 30 hotels):**

| Likely Blocker                                | Fix                                                                                      |
| --------------------------------------------- | ---------------------------------------------------------------------------------------- |
| "I don't know how to set it up properly"      | Dedicated BE setup session (30 min)                                                      |
| "I set it up but no one books through it"     | Help them promote it — website widget, Google Hotel Ads, social media                    |
| "I don't trust it — what about overbookings?" | Show sync health dashboard, zero overbooking commitment                                  |
| "My website doesn't get traffic"              | Help with basic SEO, Google Business Profile optimization, "Book Direct" messaging       |
| "I don't see the value"                       | Show them the math: "You paid ₹[X] to OTAs last month. Direct bookings = ₹0 commission." |

**Activation nudge sequence:**

| Day    | Channel  | Content                                                                                                                                               |
| ------ | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| Day 0  | Email    | "Your Booking Engine is ready — but it's not working for you yet. Here's how to activate it in 15 minutes." + setup guide link                        |
| Day 3  | CSM Call | "I noticed your Booking Engine isn't getting bookings yet. Can I help you set it up properly? It's a 30-minute session and I'll do most of the work." |
| Day 7  | Email    | "How [Hotel X] got 25 direct bookings in their first month — and saved ₹12,000 in OTA commissions"                                                    |
| Day 10 | WhatsApp | "Quick tip: Add a 'Book Direct & Save 10%' button on your website. Hotels that do this see 3x more direct bookings."                                  |
| Day 14 | CSM Call | Follow-up: "Did you get a chance to set up the BE? Let me help you right now on this call."                                                           |

**Target:** 50% of inactive BE users activated within 60 days (150 hotels)

## PHASE 3: HELP HOTELS GET MORE DIRECT BOOKINGS (Q2–Q3)

Getting BE set up is step 1. Helping hotels actually drive traffic to their direct booking page is step 2.

### Direct Booking Toolkit for Hotels

Create a "Direct Booking Starter Kit" that CSMs share with every BE-active hotel:

| Resource                        | What It Is                                                                     |
| ------------------------------- | ------------------------------------------------------------------------------ |
| Website booking widget          | Embeddable "Book Direct" button for their website                              |
| Google Business Profile guide   | Step-by-step: how to add direct booking link to Google Maps listing            |
| "Book Direct & Save" tent cards | Printable cards for hotel reception: "Book direct on our website and save 10%" |
| Social media templates          | Ready-to-post Instagram/Facebook graphics promoting direct booking             |
| Email template for past guests  | "Book directly with us next time and get [X]% off"                             |
| WhatsApp broadcast template     | Message to send to past guest contacts promoting direct booking                |
| QR code generator               | QR code linking to their booking page — for print materials, room cards, etc.  |

**Discussion Point:** Can we build these into the product as a self-serve toolkit? Or do CSMs distribute manually?

---

### Direct Booking Incentive Ideas for Hotels to Offer Guests

Help hotels understand that offering a small discount for direct bookings is still cheaper than OTA commissions:

| Strategy                           | Hotel Cost   | vs OTA Commission (18–25%) | Net Saving         |
| ---------------------------------- | ------------ | -------------------------- | ------------------ |
| 5% direct booking discount         | 5%           | 18–25%                     | 13–20% saved       |
| 10% direct booking discount        | 10%          | 18–25%                     | 8–15% saved        |
| Free breakfast for direct bookings | ~₹200/guest  | 18–25% of ₹2,000+ ADR      | Significant saving |
| Loyalty points for direct bookings | Minimal cost | 18–25%                     | Almost all saved   |

**Key message to hotels:** "Even if you give guests a 10% discount for booking direct, you're still saving 8–15% compared to OTA commissions. And you own the guest relationship."

## PHASE 4: DIRECT BOOKING ANALYTICS & VALUE VISIBILITY (Q2–Q3)

Build a dashboard every hotel can see in their account:

| Metric                               | Display                                   |
| ------------------------------------ | ----------------------------------------- |
| Total bookings this month            | Number                                    |
| Direct bookings this month           | Number + % of total                       |
| OTA bookings this month              | Number + % of total                       |
| Estimated OTA commission paid        | ₹ amount                                  |
| Commission saved via direct bookings | ₹ amount (highlighted in green)           |

**Why this matters:** When a hotel sees "You saved ₹15,000 in OTA commissions this month through direct bookings," they will never leave our platform. This is the ultimate retention tool.

## REVENUE MODEL

| Metric                                               | Conservative | Target     | Optimistic |
| ---------------------------------------------------- | ------------ | ---------- | ---------- |
| Hotels with active BE                                | 500          | 800        | 1,200      |

| Avg booking value (ADR)                              | ₹2,000       | ₹2,500     | ₹3,000     |
| Commission saved per booking (20%)                   | ₹400         | ₹500       | ₹600       |
| Total commission saved/month (all hotels)            | ₹20L         | ₹80L       | ₹2.16 Cr   |
| Our revenue opportunity (platform fee / success fee) | ₹5–10L/mo    | ₹20–40L/mo | ₹50L+/mo   |

**Discussion Point:** Do we charge a success fee on direct bookings (e.g., 3–5% vs OTA's 18–25%)? Or is the value captured purely through higher-tier subscriptions? This is a critical business model decision.
