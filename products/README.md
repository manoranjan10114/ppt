## What to Add

Drop any of these into the relevant product folder:

- Feature lists / specs
- Screenshots or screen recordings
- API documentation
- User guides
- Competitive comparisons

## What We'll Analyse

Once docs are in, we'll work through:

1. Which features belong in which pricing tier (Basic / Growth / Premium)
2. Which features should be paid add-ons
3. What usage patterns trigger upsell opportunities
4. How each product contributes to ARPU growth
5. Competitive gaps and differentiators
